# Онлайн-табло Перелёта

## Stack:
- React
- TypeScript
- Styled Components

## Для начала работы необходимо:
1. Сохранить проект локально: 
```bash
git clone https://github.com/yazakhariya/FlightBoard.git
```  
2. Ввести в консоли команду 
```bash 
npm run dev
```
